# AutoLux - Car Dealership Website

AutoLux is a front-end car dealership website designed for showcasing vehicles, services, blog posts, and customer contact options. It provides a clean UI, responsive design, and interactive navigation for users.

## Features

* Home page with hero section
* Featured cars section
* Car details pages
* Services page
* Blog page with sidebar
* Individual blog post pages
* About page
* Contact form with success popup
* Smooth hover effects
* Active navigation underline
* Open blog posts and car details in new tabs

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

AutoLux/

## Note

* CSS is written inside HTML files using internal styling.
* JavaScript is written inside HTML files using internal scripting.
* Images are loaded from online sources (Unsplash URLs).

## How to Run

1. Download or clone this repository.
2. Open `index.html` in your browser.
